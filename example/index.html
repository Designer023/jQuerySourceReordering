jQuerySourceReordering

The panels will reorder depending on their data attributes. A data-order='5' will move the panel to the position regardless of window width. A data-order='{ "468-720": 0, "1024-1200": 99 }' will only move a panel when the matching range is applicable. The last range that applies takes precidence.

A few notes for the second example are: it must be valid JSON - the quote marks must be the right type... use a 'single' quote to wrap the data-order, and "double" to wrap the ranges. Use jsonlint.com to check your JSON if you're not sure.

The plugin initialization takes 1 required argument which is the class of the item to reorder. The second argument is the option list which is optional. It currently has one vale you can change - resize. It controls whether the items should be reordered on a window size change. As this can be jarring for users it is defaulted to false. Ouside a range the order will default to the original source order.

This is ideally used in conjunction with css columns so important content can be pushed up the page. This should mean that the markup follows both a good technical and presentational hierachy and order.
